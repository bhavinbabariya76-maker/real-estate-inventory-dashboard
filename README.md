# real-estate-inventory-dashboard
Python-based inventory analytics dashboard for real estate projects using Pandas and Matplotlib.
Real Estate Inventory Dashboard

Project Overview

This project analyzes inventory performance across multiple real estate projects using Python, Pandas, and Matplotlib.

The goal is to help developers understand:

* Remaining inventory
* Project-wise sales performance
* Inventory absorption rates
* Slow-moving projects
* City-wise inventory trends

Dataset

The dataset contains information about:

* Project Name
* City
* Total Units
* Units Sold
* Average Price
* Launch Month

Key Metrics

Inventory Left

Calculates the number of unsold units for each project.

Absorption Rate

Measures the percentage of inventory sold.

Units Sold

Tracks sales performance across projects.

Business Questions Answered

* Which project has sold the most units?
* Which project has the highest unsold inventory?
* Which city has the highest inventory?
* Which projects are selling slowly?
* Which projects have the highest absorption rate?

Tools Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

Key Insights

* Identified slow-moving projects with low absorption rates.
* Highlighted projects with high remaining inventory.
* Compared project performance across multiple cities.
* Generated actionable insights for inventory planning.
